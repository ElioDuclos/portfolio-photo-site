# portfolio-photo-site

A photography portfolio website built with React, TypeScript, and Vite.

## Deployment

This site is configured to deploy automatically to GitHub Pages.

### How to enable GitHub Pages deployment

1. Go to your repository settings on GitHub
2. Navigate to **Pages** (in the left sidebar under "Code and automation")
3. Under **Build and deployment**:
   - Set **Source** to "GitHub Actions"
4. Once you merge changes to the `main` branch, the site will automatically deploy
5. Your site will be available at: `https://elioduclos.github.io/portfolio-photo-site/`

### Manual deployment

The deployment workflow can also be triggered manually:
1. Go to the **Actions** tab in your repository
2. Select the "Deploy to GitHub Pages" workflow
3. Click "Run workflow"

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Lint code
npm run lint
```
